# Sneaker-Heads

Sneaker-Heads is a responsive, multi-page front-end website built using HTML, CSS, and JavaScript. The platform showcases sneaker collections, featured styles, and brand elements in a modern, visually engaging layout.

The project focuses on clean UI design, intuitive navigation, and aesthetic presentation tailored for sneaker enthusiasts.

---

## Live Demo

🔗 [View Sneaker-Heads Live](https://sneaker-heads.vercel.app)

---

## Features

- Multi-page structure:
  - Home / Landing Page
  - About Page
  - Collections Showcase
  - Register & Form Pages
- Responsive layout for various screen sizes
- Visually rich sections with background imagery
- Clean and minimal UI focused on sneaker culture
- Lightweight and fast — no build step required
- Pure HTML, CSS & JavaScript (no backend required)
